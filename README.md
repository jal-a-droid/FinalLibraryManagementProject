This program was created by the students of the class of 2026 at high tech los angeles high school.
this program was created by Jalen Abraham, Vincent Gale, Javier Sanabria, and Ethan Malka.

This program is simply a library management system built on java, and it contains ~7 classes including the main
java class. You can add, edit, delete, and update statuses of books through the program. You can also add different accounts
if the same program is used on the same computer by multiple people. the program we have currently as of 6/1/26,
is fully working without any bugs which is great. currently there is a test account with some other modifiable data
that can be messed around with, since this program is just a demonstration/project for our class.
